File description

FireyCallouts is a callout pack for LSPDFR which features several callouts themed around the cooperation with the available emergency services in GTA V.
For more information visit the FireyCallouts thread on the LSPDFR website. You can also download the latest stable version there.

If you are interested in experimental versions of the callout pack you can download it from this website. The versions available here might not be stable and cause crashes to your game.
If you encounter any bugs or crashes with the callout pack consider opening an Issue in the Github repository. Otherwise use the LSPDFR Forum as means of communication.
